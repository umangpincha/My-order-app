# Raja Order Book 📋

Professional mobile-friendly order book application.

## Features
- ✅ Login system (ID: RAJA, Password: 54321)
- ✅ Product categories with items side-by-side
- ✅ Quantity + Rate input per item with auto-calculation
- ✅ Gross Qty / Items / Value shown in bold header
- ✅ Export as PDF (professional layout)
- ✅ Share via WhatsApp
- ✅ Export as Excel
- ✅ Save favourite orders
- ✅ Notes/message section (appears in PDF & WhatsApp)
- ✅ Mobile-first PWA design
- ✅ Popular items appear first

## Deploy to Vercel
1. Push to GitHub
2. Connect repo on [vercel.com](https://vercel.com)
3. Deploy — done!

## Run Locally
```bash
npm install
npm run dev
```
